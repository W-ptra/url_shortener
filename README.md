# Description
A simple URL shortener by hashing the original url using md5 and took the first 5 letter and store it on mysql

# requirement
1. node.js & npm
2. mysql

# how to use
1. Clone repository
``git clone https://github.com/W-ptra/url_shortener.git``
2. Run
`` node . ``
3. Open browser
insert url ``http://localhost:8080``
